# Clima Garantido - Landing Page

Esta é a landing page da **Clima Garantido**, adaptada para ser executada diretamente no **GitHub Pages**.

## Como publicar no GitHub Pages

Para colocar este site no ar, siga estes passos simples:

1. **Crie um novo repositório** no seu GitHub (ex: `clima-garantido`).
2. **Faça o upload** do arquivo `index.html` para a raiz desse repositório.
3. No GitHub, vá em **Settings** (Configurações) do seu repositório.
4. No menu lateral esquerdo, clique em **Pages**.
5. Em **Build and deployment** > **Branch**, selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
6. Clique em **Save**.
7. Aguarde alguns minutos e o GitHub fornecerá o link público do seu site (ex: `https://seu-usuario.github.io/clima-garantido/`).

## Arquivos incluídos
- `index.html`: O arquivo principal da landing page (renomeado para que o GitHub o reconheça automaticamente como a página inicial).
